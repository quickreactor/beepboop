# Security Policy

## Supported Versions

Currently supported versions: 1.0.0 and later

## Reporting Vulnerabilities

If you discover a security vulnerability, please email it to: beepboop-security@example.com

**Do not create public issues for security vulnerabilities.**

## What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if known)
- Your name/handle for credit (optional)

## Response Process

I will acknowledge receipt within 48 hours and provide regular updates on the fix status.

## Security Best Practices

- Always grant BeepBoop only the permissions it needs
- Review permission requests carefully
- Keep the app updated to the latest version
- Report suspicious behavior via email
